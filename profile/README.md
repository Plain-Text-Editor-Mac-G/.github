## 🔍 Overview

<a href="#plainTextEditor">Plain Text Editor</a> is a deliberately minimal text editor for Mac — providing a clean, distraction-free environment for writing and editing plain text without formatting options, rich text modes, toolbar buttons, or any interface element that isn't directly relevant to typing and editing words.

The <a href="#plainTextEditor">plain text enforcement</a> is the application's core principle — Plain Text Editor accepts only plain text and produces only plain text. There is no way to make text bold, change fonts within a document, add colors, or create any formatting that goes beyond what a plain .txt file can contain. When text is pasted from a formatted source — a web page, a word processor document, an email — formatting is automatically stripped, leaving only the text content. This stripping behavior makes Plain Text Editor the fastest way to remove formatting from copied text on Mac. The <a href="#plainTextEditor">distraction-free interface</a> provides exactly what plain text editing requires — a text area, the text, and nothing else visible unless specifically requested. No toolbar, no sidebar, no formatting options, no style controls. The interface communicates clearly that the application's purpose is text, not text formatting.

The <a href="#plainTextEditor">monospaced font display</a> presents text in a monospaced typeface by default — appropriate for code snippets, configuration files, and structured text where character alignment matters, and readable for prose writing in a way that many writers prefer for the visual distinction from formatted document writing. The <a href="#plainTextEditor">line numbers</a> display the current cursor position and line count alongside the text — useful for code editing, for finding specific lines in structured files, and for orientation in longer documents.

<a href="#plainTextEditor">Word and character count</a> displays the current document's word and character count — useful for writing within specific length requirements, tracking progress toward length goals, and any context where text quantity matters. <a href="#plainTextEditor">Dark mode support</a> follows the macOS system appearance setting automatically — providing a dark background text editing environment in dark mode without manual configuration.

<a href="#plainTextEditor">Font size customization</a> adjusts the text display size to match reading and writing preferences — larger for comfortable reading, smaller for seeing more content, without affecting the saved file content. <a href="#plainTextEditor">Sindre Sorhus development quality</a> from one of the most prolific and respected independent Mac application developers — a track record of thoughtful, well-maintained small applications that do one thing well and continue receiving updates as macOS evolves.
